# ML Gift Random Pick - Web

**Descripción**
ML Gift Random Pick es una aplicación web que permite obtener un artículo de Mercado Libre bajo ciertos parámetros.  
La idea es que el usuario configure filtros como:
- Palabras clave  
- Índices de precios  
- Si es Full o no  
- Si es extranjero o no  
- Otros parámetros adicionales  

Una vez configurado, el usuario pulsa un botón y aparece una especie de ruleta que selecciona aleatoriamente el regalo que le toca.

**Funcionalidades**
- Búsqueda de artículos en Mercado Libre según filtros definidos.  
- Visualización del artículo seleccionado: título, precio, imagen y enlace a Mercado Libre.  
- Experiencia interactiva tipo ruleta para la selección del regalo.  

**Tecnologías**
- Frontend: HTML, CSS, JavaScript 
- Backend: API REST
